# 🐾 Proyecto Veterinaria - Base de Datos

¡Bienvenido al sistema de gestión de mi Veterinaria!

Este proyecto incluye una base de datos completa diseñada en **MySQL** corriendo sobre **Docker**

## 📂 Contenido del Repositorio
- **`veterinaria.sql`**: Contiene toda la estructura (tablas) y los datos de prueba.
- **Diagrama ER**: Puedes encontrar el diseño lógico de las tablas en el archivo.
## 🛠️ Tecnologías usadas
- Docker & Docker Compose 
- MySQL 
- HeidiSQL 
- DrawDB para el diseño
